   - 선택된 필터에 따라 해당 팀의 [매출 추이, 인원 대비 매출 성과] 차트가 동적으로 업데이트되는 '팀별 대시보드'를 구성하세요.

3. 데이터 입력 모달 (Harness Input):
   - 우측 상단에 '기초데이터 입력' 버튼을 생성하고, 클릭 시 모달(Modal) 창을 띄우세요.
   - 모달 내부에는 CSV 파일을 드래그 앤 드롭으로 업로드할 수 있는 영역을 만드세요.

4. 데이터 파싱 및 반영 로직:
   - 업로드된 CSV를 파싱하여 현재 대시보드의 상태(State)에 즉시 반영하는 하네스 로직을 작성하세요.
   - 데이터 반영 시 차트가 부드럽게 업데이트되도록 애니메이션을 적용하세요.

5. 격리 가이드:
   - 이 모든 작업은 메인 대시보드 소스 파일을 직접 수정하지 말고, `src/features/detail` 폴더 내에 별도의 컴포넌트로 독립적으로 구현하세요.
